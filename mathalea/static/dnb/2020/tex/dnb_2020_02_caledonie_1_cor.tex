\textbf{Affirmation 1} :

$f(5) = 2\times (5 - 3) = 2 \times 2 = 4$. L'affirmation est vraie.

\medskip

\textbf{Affirmation 2} :

On a $84 \times \np{256000} = \np{21504000}$~(W) $= \np{21,504}$~MW soit environ 21,5~MW. L'affirmation est vraie.

\textbf{Affirmation 3} :

On a $\dfrac{\text{EA}}{\text{ED}} = \dfrac{2,8}{1,2} = \dfrac{28}{12} = \dfrac{7}{3}$ ;

$\dfrac{\text{EB}}{\text{EC}} = \dfrac{3,4}{1,6} = \dfrac{34}{16} = \dfrac{17}{8}$.

Or $\dfrac{7}{3} \ne  \dfrac{17}{8}$ car $7 \times 8 \ne 3 \times 17$.

L'affirmation est fausse.

\medskip

\textbf{Affirmation 4} :

Si $\alpha$ est la mesure commune aux cinq angles, on a $5\alpha = 360$ soit $\alpha = \dfrac{360}{5} = \dfrac{720}{10} = 72(\degres{})$.

L'affirmation est fausse.

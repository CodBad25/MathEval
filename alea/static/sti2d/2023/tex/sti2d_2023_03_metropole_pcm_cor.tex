\begin{enumerate}[start=4]
\item Les solutions de l'équation différentielle : $\dfrac{\text{d}v}{\text{d}t} + 6,8v = 0$, sont définies par :

$v(t) = K\text{e}^{-6,8t}$, avec $K \in \R$.

Une solution particulière constante de cette équation est donnée par :
\[\dfrac{\text{d}v}{\text{d}t} = 0 \iff  - 6,8v + 7,5 = 0 \iff v = \dfrac{7,5}{6,8} = \dfrac{75}{68}.\]

La solution générale de cette équation différentielle est donc donnée par : 
\[v(t) = K\text{e}^{-6,8t} + \dfrac{75}{68},\]
et comme :
\[v(0) = 0 \iff K + \dfrac{75}{68} = 0 \iff K = - \dfrac{75}{68},\]
soit finalement :
\[v(t) = - \dfrac{75}{68}\text{e}^{-6,8t} + \dfrac{75}{68}.\]

\item On sait que $\displaystyle\lim_{t \to + \infty} \text{e}^{-6,8t} = 0$ et également que $\displaystyle\lim_{t \to + \infty} - \dfrac{75}{68}\text{e}^{-6,8t} = 0$, donc par somme de limites $\displaystyle\lim_{t \to + \infty} = v(t) = v_{\text{lim}} = \dfrac{75}{68}$.

\item On calcule $\eta = \dfrac{\left(20,1\cdot 10^{-3} - 840 \times 5,6 \cdot 10^{-6}\right)\times 9,81}{\left(6\pi \times 0,011 \times 1,1\right)} \approx 0,66$~kg.m$^{-1}$.\,s$^{-1}$.

C'est à peu près la valeur fournie par le fabricant.
\end{enumerate}

On résout cette équation:
$\e^{-0,027 t_{0,5}} = \dfrac{1}{2}
\iff
-0,027 t_{0,5} = \ln \left (\frac{1}{2}\right )
\iff
t_{0,5} = \dfrac{\ln \left (\frac{1}{2}\right )}{-0,027 }
$

Donc $t_{0,5}\approx 25,7$.

  
\medskip

Un jeu consiste à combattre en duel soit un monstre A, soit un monstre B.

On a une probabilité de $\dfrac{4}{5}$ d’affronter le monstre A.

Le joueur gagne contre le monstre A dans 30\,\% des cas, et gagne contre le monstre
B dans 25\,\% des cas.

Le joueur lance une partie. On considère les évènements :

\begin{itemize}
\item $A$ : \og Le joueur affronte le monstre A. \fg
\item $B$ : \og Le joueur affronte le monstre B. \fg
\item $V$ : \og Le joueur est victorieux. \fg
\end{itemize}

\medskip

\begin{enumerate}
\item Déterminer $p_B\left(\overline{V}\right)$ et interpréter le résultat.
\item Montrer que $p(B \cap V) = \dfrac{1}{20}$.
\item Calculer $p(V)$.
\item Calculer la probabilité d’avoir combattu le monstre B sachant que le joueur est victorieux.
\end{enumerate}

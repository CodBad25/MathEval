\subsection*{Question 4}

L'inéquation peut s'écrire :
\[
-3x^2 + 11x + 4 \leqslant 0.
\]
Pour le trinôme \(-3x^2 + 11x + 4\), on a :
\[
\Delta = 11^2 + 4 \times 3 \times 4 = 121 + 48 = 169 = 13^2 > 0.
\]
Le trinôme a donc deux solutions :
\[
x_1 = \dfrac{-11 + 13}{-6} = -\dfrac{1}{3} \quad \text{et} \quad x_2 = \dfrac{-11 - 13}{-6} = 4.
\]
On sait que le trinôme est du signe de \(a = -3 < 0\), sauf entre les racines.

On a donc :
\[
S = \left]-\infty \,;\, -\dfrac{1}{3} \right] \cup [4 \,;\, +\infty[.
\]

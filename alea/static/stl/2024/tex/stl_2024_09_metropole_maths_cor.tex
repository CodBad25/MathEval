\textbf{Question 2}

\medskip

On applique la fonction exponentielle de chaque côté :
\[\ln(2x + 1) = 7 \Rightarrow 2x + 1 = e^7 \Rightarrow 2x = e^7 - 1 \Rightarrow x = \dfrac{e^7 - 1}{2}.\]

On a $x \approx 547,8$, ce qui est bien dans l'intervalle $\left] - \dfrac{1}{2}~;+\infty \right[$.

\textbf{Question 4}

\medskip

\[\int_0^1 f(x)\,dx = F(1) - F(0)\]
Comme : $F(x) = x^3 + 7x$
\[F(1) = 1^3 + 7 \times 1 = 8, \quad F(0) = 0^3 + 7 \times 0 = 0 \Rightarrow \int_0^1 f(x)\,dx = 8 - 0 = 8\]
